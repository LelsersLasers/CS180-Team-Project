
## Overview

- User:
	- id
	- username
	- password
	- picture (base64 string)
	- [friends]
	- [blocked]
- Post:
	- id
	- [user]
	- picture (base64 string)
	- text
	- date
	- [upvotes]
	- [downvotes]
	- [comments]
- Comment:
	- id
	- [user]
	- [post]
	- text
	- [upvotes]
	- [downvotes]

## As tables

- User:
	- id: int
	- username: String
	- password: String
	- picture: String (base64)
- Post:
	- id: int
	- userId: User.id
	- picture: String (base64)
	- text: String
	- date: String
- Comment:
	- id: int
	- userId: User.id
	- postId: Post.id
	- text: String
- Friendship:
	- id: int
	- user1Id: User.id
	- user2Id: User.id
- Blocked:
	- id: int
	- blockerId: User.id
	- blockedId: User.id
- PostVote:
	- id: int
	- userId: User.id
	- postId: Post.id
	- vote: int (1 for upvote, -1 for downvote)
- CommentVote:
	- id: int
	- userId: User.id
	- commentId: Comment.id
	- vote: int (1 for upvote, -1 for downvote)
